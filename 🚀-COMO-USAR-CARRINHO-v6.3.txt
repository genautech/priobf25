🚀 COMO USAR O CARRINHO WHATSAPP - v6.3
========================================

📱 GUIA RÁPIDO PARA O CLIENTE
=============================

1️⃣ SELECIONAR PRODUTO
----------------------
→ Navegue pelo catálogo
→ Clique em "Adicionar à Seleção" no produto desejado
→ Controles de quantidade aparecerão


2️⃣ AJUSTAR QUANTIDADE
----------------------
→ Use os botões [+] e [-] para mudar quantidade
→ Ou digite direto no campo numérico
→ Investimento atualiza automaticamente
→ Estatísticas no topo também atualizam


3️⃣ ENVIAR PEDIDO (OPÇÃO 1: PRODUTO ÚNICO)
------------------------------------------
→ Clique no botão verde "🛒 Enviar ao Carrinho"
→ Veja confirmação com resumo do pedido
→ Confirme clicando "OK"
→ WhatsApp abre automaticamente com mensagem pronta
→ Revise e envie para o vendedor


3️⃣ ENVIAR PEDIDO (OPÇÃO 2: MÚLTIPLOS PRODUTOS)
-----------------------------------------------
→ Selecione vários produtos
→ Ajuste quantidade de cada um
→ Clique no botão global "🛒 Enviar Todos ao Carrinho"
   (aparece acima dos filtros)
→ Veja confirmação com todos produtos
→ Confirme clicando "OK"
→ WhatsApp abre com mensagem completa
→ Revise e envie para o vendedor


4️⃣ REMOVER PRODUTO DA SELEÇÃO
------------------------------
→ Produto selecionado tem dois botões:
  • Verde: "Enviar ao Carrinho"
  • Vermelho: "Remover"
→ Clique no botão vermelho para remover
→ Produto volta ao estado normal
→ Quantidade é limpa

⚙️ CONFIGURAÇÃO PARA ADMINISTRADOR
===================================

⚠️ IMPORTANTE: CONFIGURE O WHATSAPP ANTES DE USAR!
---------------------------------------------------

1. Abra o arquivo: cliente-analise-v6.3.html

2. Procure por: 5541999999999 (aparece 2 vezes)

3. Substitua pelo número REAL do WhatsApp:
   
   Linha ~716:
   const numeroWhatsApp = '5541999999999'; // ← ALTERE AQUI
   
   Linha ~800:
   const numeroWhatsApp = '5541999999999'; // ← ALTERE AQUI

4. Formato correto:
   - 55 = DDI Brasil
   - DD = DDD (11, 21, 41, etc)
   - NNNNNNNNN = Número com 9 dígitos
   
   Exemplos:
   - São Paulo: '5511987654321'
   - Rio de Janeiro: '5521987654321'
   - Curitiba: '5541987654321'

5. Salve o arquivo

6. Teste localmente antes de fazer deploy

🐛 TROUBLESHOOTING
==================

Problema: WhatsApp não abre
----------------------------
→ Verifique se navegador permite pop-ups
→ Tente em outra aba/navegador
→ Verifique número do WhatsApp no código


Problema: Mensagem aparece truncada
------------------------------------
→ Normal, WhatsApp tem limite de caracteres
→ Mensagem completa estará no campo de texto
→ Cliente pode revisar antes de enviar


Problema: Número aparece como "5541999999999"
----------------------------------------------
→ Você NÃO configurou o número no código!
→ Veja seção "CONFIGURAÇÃO PARA ADMINISTRADOR"
→ Substitua nas 2 ocorrências


Problema: Botões não aparecem
------------------------------
→ Limpe cache do navegador (Ctrl+Shift+Delete)
→ Recarregue página (Ctrl+F5)
→ Verifique se salvou o arquivo corretamente
