                      COMO IDENTIFICAR OS NOVOS
═══════════════════════════════════════════════════════════════════════════

PROCURE POR:

1. Ícone 🔗 (external-link-alt) ao lado do texto do botão
2. Texto "Abrir..." no botão (ao invés de ação direta)
3. Badge "🆕" ou "EXTERNO" no canto superior direito
4. Cores diferentes: Rosa, Índigo, Lima, Céu, Âmbar

EXEMPLO DE CARD NOVO:
┌──────────────────────┐
│  🔍 🆕               │ ← Badge NOVO
│  Buscar Imagens FINAL│ ← Título claro
│                      │
│  Buscar imagens reais│ ← Descrição
│  de produtos online  │
│                      │
│ [🔗 Abrir Buscador]  │ ← Ícone external-link + texto "Abrir"
└──────────────────────┘

CLIQUE → ABRE EM NOVA ABA INSTANTANEAMENTE ✅
