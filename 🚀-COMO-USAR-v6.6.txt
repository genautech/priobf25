───────────────────────────────────────────────────────────────────

1️⃣ ABRIR O ADMIN v6.6

4️⃣ LIMPAR HISTÓRICO (OPCIONAL)
   • Expanda o widget
   • Clique em "🗑️ Limpar Histórico"
   • Confirme a ação
   • Histórico é zerado

💡 DICAS DE USO
───────────────────────────────────────────────────────────────────

✅ MANTENHA O WIDGET FECHADO
   • Widget inicia fechado para economizar espaço
   • Abra apenas quando precisar consultar

✅ VERIFIQUE O ÚLTIMO COMMIT
   • Sempre que abrir o admin, olhe a data/hora
   • Se muito antiga, pode ter problemas de sync

✅ USE EM DIFERENTES COMPUTADORES
   • Escritório: Edita de manhã, vê último commit
   • Casa: Abre à tarde, sabe que houve mudanças
   • Coordenação perfeita!

✅ HISTÓRICO LOCAL
   • Cada computador mantém seu próprio histórico
   • Não é sincronizado entre máquinas
   • Use "último commit" para saber mudanças remotas

🧪 TESTE RÁPIDO
───────────────────────────────────────────────────────────────────

───────────────────────────────────────────────────────────────────
🚨 PROBLEMAS CONHECIDOS
───────────────────────────────────────────────────────────────────

❌ "Último commit" não carrega
   → Verifique se GitHub está configurado
   → Token precisa ter permissão 'repo'
   → Console (F12) mostra erros?

❌ Histórico não aparece após editar
   → localStorage habilitado?
   → Modo privado/anônimo pode bloquear
   → Tente em janela normal

❌ Horário errado
   → Timezone do sistema operacional
   → Sempre mostra America/Sao_Paulo (UTC-3)
   → Curitiba/PR, Brasil

───────────────────────────────────────────────────────────────────
📊 ESTRUTURA DO HISTÓRICO
───────────────────────────────────────────────────────────────────

Cada registro contém:
• ID único (timestamp)
• ID do produto
• Nome do produto
• Ação (criado/editado/duplicado)
• Timestamp (milissegundos)
• Data formatada (DD/MM/AAAA HH:MM:SS)

Armazenamento: localStorage
Key: 'productUpdateHistory'
Limite: 50 registros (mais antigos removidos automaticamente)
