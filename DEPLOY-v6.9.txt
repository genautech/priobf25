⚠️ EM CASO DE ERRO:

ERRO: "Changes not staged for commit"
SOLUÇÃO: Execute git add novamente

ERRO: "Nothing to commit"
SOLUÇÃO: Verifique se os arquivos foram modificados

ERRO: "Permission denied"
SOLUÇÃO: Verifique suas credenciais do GitHub

ERRO: "Rejected - non-fast-forward"
SOLUÇÃO: git pull origin main --rebase
         depois: git push origin main
