═══════════════════════════════════════════════════════════════════
  🎯 COMO USAR
═══════════════════════════════════════════════════════════════════

PASSO 1: Acessar Admin
  → Abrir admin-v6.1.html
  → Clicar na aba "Ferramentas"

PASSO 2: Abrir Importador
  → Clicar no botão "Importador Avançado"
  → Nova aba abre automaticamente

PASSO 3: Ver Instruções
  → Notificação aparece com guia completo
  → Lista todos os recursos
  → Explica passo a passo

PASSO 4: Usar o Importador
  → Cole seu JSON na área de texto
  → Clique em "Converter JSON"
  → Revise o preview
  → Copie o código gerado
  → Cole no produtos-v6.1.js
