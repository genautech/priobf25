═══════════════════════════════════════════════════════════════
  🚀 COMO USAR (4 PASSOS)
═══════════════════════════════════════════════════════════════

PASSO 1: OBTER API KEY (5 minutos - só primeira vez)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Acesse: https://serpapi.com/users/sign_up
2. Cadastre-se (GRÁTIS, sem cartão)
3. Confirme email
4. Copie API Key em: https://serpapi.com/manage-api-key
5. Salve a Key para reusar


PASSO 2: CONFIGURAR (10 segundos)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Abra: buscar-imagens-google.html
2. Escolha: "SerpAPI"
3. Cole: Sua API Key


PASSO 3: PROCESSAR (20-25 minutos)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Clique: [📦 Carregar Produtos]
   → 142 produtos carregados

2. Clique: [🚀 Buscar Todas]
   → Busca 100 produtos automaticamente (5 min)
   → Plano grátis: 100 buscas/mês

3. Para os 42 restantes:
   → Clique [🔍 Buscar] em cada um
   → Escolhe manualmente a melhor imagem
   → Ou clique [✏️ Manual] e cole URL
   → 30s cada = 20 minutos


PASSO 4: EXPORTAR (5 segundos)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Clique: [💾 Exportar JSON]
2. Arquivo: produtos-imagens-[data].json
3. Pronto! ✅
