      │     → ATUALIZAR produto existente   │
      │     → Manter ID original            │
      │     → Aplicar novos valores         │
      │                                      │
      │ 2b. SE NÃO ENCONTRADO:              │
      │     → ADICIONAR como novo produto   │
      │     → Usar ID do arquivo            │
      │     → Incrementar total             │
      └──────────────────────────────────────┘
      
   4️⃣ RESULTADO
      ↓
      ✅ Catálogo atualizado na memória
      ✅ Mudanças visíveis em todas as abas
      ✅ Mensagem: "X atualizados, Y novos"

════════════════════════════════════════════════════════════════
   ⚠️ CUIDADOS IMPORTANTES
════════════════════════════════════════════════════════════════

✅ FAZER:
   • Sempre exportar backup antes de importar
   • Validar JSON antes de importar
   • Testar com poucos produtos primeiro
   • Usar IDs únicos para novos produtos
   • Manter arquivo de backup datado

❌ NÃO FAZER:
   • Importar sem backup
   • Usar IDs duplicados
   • Editar JSON sem validar sintaxe
   • Confiar 100% sem testar antes
   • Deletar todos os backups

⚠️ LIMITAÇÕES:
   • Mudanças ficam apenas na MEMÓRIA
   • Recarregar página = perder alterações
   • Para salvar permanente = atualizar produtos-v6.1.js
