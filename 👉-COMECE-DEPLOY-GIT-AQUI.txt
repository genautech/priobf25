Se for a PRIMEIRA VEZ ou teve erro "Permission denied":

OPÇÃO 1 (Mais Fácil): GitHub CLI
  1. Instale: https://cli.github.com/
     Windows: winget install --id GitHub.cli
     Mac: brew install gh
     Linux: sudo apt install gh
  
  2. Faça login:
     gh auth login
     └─ Escolha: GitHub.com
     └─ Escolha: HTTPS
     └─ Escolha: Login with a web browser
     └─ Autorize no navegador
  
  3. ✅ PRONTO! Pode executar os comandos acima

OPÇÃO 2: Token Pessoal
  1. Crie token: https://github.com/settings/tokens
     └─ Generate new token (classic)
     └─ Marque: ✅ repo
     └─ Copie o token
  
  2. Configure:
     git config --global credential.helper store
  
  3. No primeiro push:
     Username: seu_usuario
     Password: COLE_O_TOKEN (não a senha!)
  
  4. ✅ PRONTO! Próximos pushs automáticos
