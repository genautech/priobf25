│ → Base para calcular nossa competitividade                │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ Preço Venda Yoobe (R$)                                     │
│ → Preço que vendemos na Yoobe                              │
│ → Calculado: Custo Prio + Margem                           │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ Diferença Yoobe vs Prio (R$)                               │
│ → Quanto custa a mais/menos que o fornecedor Prio          │
│ → Positivo = Yoobe mais caro                               │
│ → Negativo = Yoobe mais barato                             │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ Status Competitividade                                     │
│ → "Yoobe mais barato"   = ✅ Competitivo                  │
│ → "Preço igual"         = ⚠️ Neutro                       │
│ → "Yoobe mais caro"     = ❌ Não competitivo              │
│ → "Sem comparação"      = ℹ️ Sem preço Prio               │
└────────────────────────────────────────────────────────────┘

   💡 ANÁLISES ÚTEIS NO EXCEL
════════════════════════════════════════════════════════════════

1️⃣ TAXA DE COMPETITIVIDADE
   Contar por "Status Competitividade":
   
   Yoobe mais barato:  15 produtos (25%)  ✅
   Preço igual:         5 produtos  (8%)  ⚠️
   Yoobe mais caro:    25 produtos (42%)  ❌
   Sem comparação:     97 produtos (68%)  ℹ️

2️⃣ ECONOMIA TOTAL POSSÍVEL
   Somar "Diferença" onde "Yoobe mais caro"
   = Quanto economizamos se comprar do Prio
   
   Exemplo: R$ 3.500,00 de economia possível

3️⃣ PRODUTOS PRIORITÁRIOS
   Filtrar:
   - Status = "Yoobe mais caro"
   - Diferença > R$ 100
   - Quantidade > 20
   
   = Produtos onde temos prejuízo significativo

4️⃣ TOP 10 MAIORES ECONOMIAS
   Ordenar por "Diferença" (maior → menor)
   Ver top 10 produtos
   = Onde devemos focar para reduzir custos
