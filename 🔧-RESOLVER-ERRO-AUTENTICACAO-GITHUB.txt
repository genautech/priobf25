ERRO:
remote: Invalid username or token. Password authentication is not 
supported for Git operations.
fatal: Authentication failed for 'https://github.com/genautech/priobf25.git/'

CAUSA:
GitHub não aceita mais senha, precisa usar Personal Access Token (PAT)

───────────────────────────────────────────────────────────────────
✅ SOLUÇÃO 1: CRIAR NOVO TOKEN (RECOMENDADO)
───────────────────────────────────────────────────────────────────

PASSO 1: Criar Personal Access Token

1. Abrir: https://github.com/settings/tokens

2. Clicar em "Generate new token" → "Generate new token (classic)"

3. Configurar:
   Nome: "priobf25-deploy"
   Expiração: 90 days (ou No expiration)
   
4. Marcar APENAS estes scopes:
   ✅ repo (todos os sub-itens)
   
5. Clicar em "Generate token"

6. COPIAR O TOKEN (guarde em local seguro!)
   Exemplo: ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

───────────────────────────────────────────────────────────────────
PASSO 2: Configurar o Token no Mac
───────────────────────────────────────────────────────────────────

OPÇÃO A: Atualizar credencial no Keychain (Mac)

1. Abrir "Keychain Access" (Acesso ao Chaveiro)
   Spotlight → digite "Keychain Access"

2. Buscar por "github.com"

3. Deletar todas as entradas antigas do GitHub

4. Tentar push novamente:
   git push origin main
   
5. Quando pedir credenciais:
   Username: genautech
   Password: [COLAR O TOKEN AQUI]

───────────────────────────────────────────────────────────────────
OPÇÃO B: Usar SSH em vez de HTTPS (mais seguro)
───────────────────────────────────────────────────────────────────

1. Verificar se já tem SSH key:
   ls -al ~/.ssh
   
2. Se não tiver, criar nova:
   ssh-keygen -t ed25519 -C "seu-email@example.com"
   (Pressionar Enter em tudo)
   
3. Copiar a chave pública:
   cat ~/.ssh/id_ed25519.pub | pbcopy
   
4. Adicionar no GitHub:
   https://github.com/settings/keys
   → New SSH key
   → Colar a chave
   → Add SSH key
   
5. Mudar remote para SSH:
   git remote set-url origin git@github.com:genautech/priobf25.git
   
6. Testar:
   git push origin main

───────────────────────────────────────────────────────────────────
✅ SOLUÇÃO 2: USAR GITHUB DESKTOP (MAIS FÁCIL)
───────────────────────────────────────────────────────────────────

1. Baixar GitHub Desktop:
   https://desktop.github.com/
   
2. Instalar e fazer login

3. Abrir repositório:
   File → Add Local Repository
   → Selecionar pasta "priobf25"
   
4. Fazer commit e push pela interface gráfica
   (Muito mais fácil!)

───────────────────────────────────────────────────────────────────
✅ SOLUÇÃO 3: USAR TOKEN DIRETO NA URL (TEMPORÁRIO)
───────────────────────────────────────────────────────────────────

⚠️ NÃO RECOMENDADO (menos seguro)

git remote set-url origin https://SEU_TOKEN@github.com/genautech/priobf25.git

Substitua SEU_TOKEN pelo token que você criou.

Depois fazer:
git push origin main

SOLUÇÃO:
1. Abrir Terminal
2. Executar:

   protocol=https
