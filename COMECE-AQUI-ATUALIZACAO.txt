📊 MAPEAMENTO DE CAMPOS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

┌─────────────────────────────┬─────────────────────────────────┐
│ CAMPO CSV                   │ CAMPO SISTEMA                   │
├─────────────────────────────┼─────────────────────────────────┤
│ SKU                         │ sku                             │
│ Nome do Produto             │ nome                            │
│ Categoria                   │ categoria                       │
│ Subcategoria                │ subcategoria                    │
│ Fornecedor                  │ fornecedor                      │
│ Preço Mercado               │ custoBase, precoMercado         │
│ Menor preço Marketplaces    │ precoVenda (ou calculado)       │
│ Link de compra              │ linkCompra                      │
│ Marketplace                 │ especificacoes.marketplace      │
│ Prazo de entrega            │ especificacoes.prazoEntrega     │
└─────────────────────────────┴─────────────────────────────────┘

CAMPOS AUTOMÁTICOS:
• id: Auto-incremento (1001, 1002, 1003...)
• quantidade: 10 (padrão)
• margem: Calculado automaticamente (padrão 30%)
• imagem: Placeholder temporário
• estoque: "Disponível" ou "A Definir"


⚠️ PONTOS DE ATENÇÃO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. PRODUTOS SEM PREÇO
   Situação: Alguns produtos têm campo "Preço Mercado" vazio
   Resultado: Produto criado com custoBase = 0
   Ação: Preencher manualmente no Admin após importação

2. PRODUTOS SEM LINK
   Situação: Campo "Link de compra" vazio
   Resultado: Produto criado, mas sem link funcional
   Ação: Adicionar links depois via Admin (opcional)

3. IMAGENS PLACEHOLDER
   Situação: Sistema gera URLs temporárias
   Resultado: Imagens genéricas com nome do produto
   Ação: Usar ferramenta "Buscar Imagens" no Admin depois

4. VALORES COM VÍRGULA
   Situação: CSV usa "1.234,56" (formato brasileiro)
   Resultado: Conversor transforma automaticamente para "1234.56"
   Ação: Nenhuma - conversão automática!

Q: Posso testar antes de substituir?
A: Sim! Baixe o arquivo gerado, abra-o em um editor de texto e
   verifique se os dados estão corretos. Você pode fazer quantas
   conversões quiser sem afetar o sistema.

Q: E se algo der errado?
A: Sempre faça backup antes! Use:
   cp produtos-v6.1.js produtos-v6.1.js.backup
   Se precisar reverter, é só copiar o backup de volta.

Q: Os produtos sugeridos serão perdidos?
A: Sim. A planilha CSV substitui TODOS os produtos. Se você tem
   produtos sugeridos importantes, salve-os antes em um arquivo
   separado para importar depois.

Q: Quanto tempo leva?
A: Conversão: 5-10 segundos
   Deploy: 3-4 minutos após o push
   Total: ~5 minutos do início ao fim

Q: Preciso atualizar o Admin ou Cliente?
A: Não! As páginas admin-v6.10.html e cliente-analise-v6.3.5.html
   continuam funcionando perfeitamente com o novo catálogo.

Q: Os cupons de desconto são mantidos?
A: Sim! O conversor preserva todos os cupons existentes no sistema.
