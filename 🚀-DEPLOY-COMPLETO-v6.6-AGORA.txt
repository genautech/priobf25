🔧 SE HOUVER CONFLITOS
───────────────────────────────────────────────────────────────────

Se aparecer erro de conflito:

# Se conflito no produtos-v6.1.js:

git push origin main

───────────────────────────────────────────────────────────────────
🆘 SE DER ERRO NO PUSH
───────────────────────────────────────────────────────────────────

Erro: "Push rejected" ou "Behind remote"

Solução:
git pull --rebase origin main
git push origin main
