⚡ COMANDOS GIT PRONTOS - v6.6
═══════════════════════════════════════════════════════════════════

Copie e cole estes comandos quando estiver pronto para fazer deploy!

───────────────────────────────────────────────────────────────────
🚀 OPÇÃO 1: DEPLOY COMPLETO (RECOMENDADO)
───────────────────────────────────────────────────────────────────

# 1. Ir para o diretório do projeto
cd priobf25

# 2. Ver o que mudou
git status

# 3. Adicionar TODOS os arquivos
git add .

# 4. Fazer commit
git commit -m "v6.6: Histórico de atualizações + último commit (Curitiba timezone)"

# 5. Push para GitHub
git push origin main

# 6. Aguardar 3-4 minutos
# Cloudflare Pages fará deploy automático

# 7. Testar
# Abrir: https://priobf25.pages.dev/admin-v6.6.html

───────────────────────────────────────────────────────────────────
🎯 OPÇÃO 2: DEPLOY APENAS DO ADMIN (SEM DOCS)
───────────────────────────────────────────────────────────────────

# 1. Ir para o diretório
cd priobf25

# 2. Adicionar apenas arquivos principais
git add admin-v6.6.html
git add index.html
git add README.md

# 3. Commit
git commit -m "v6.6: Admin com histórico de atualizações"

# 4. Push
git push origin main

───────────────────────────────────────────────────────────────────
🧪 OPÇÃO 3: DEPLOY EXPERIMENTAL (MANTÉM v6.5 ATIVO)
───────────────────────────────────────────────────────────────────

# 1. Ir para o diretório
cd priobf25

# 2. Adicionar apenas novo admin
git add admin-v6.6.html

# 3. Commit
git commit -m "v6.6: Nova versão admin (experimental)"

# 4. Push
git push origin main

# 5. Testar
# Abrir: https://priobf25.pages.dev/admin-v6.6.html

# 6. Se funcionar bem, atualizar index.html depois:
git add index.html
git commit -m "Atualizar index para v6.6"
git push origin main

───────────────────────────────────────────────────────────────────
🔧 SE HOUVER CONFLITOS
───────────────────────────────────────────────────────────────────

# 1. Atualizar do GitHub primeiro
git pull origin main

# 2. Se houver conflitos no produtos-v6.1.js:
git checkout --theirs produtos-v6.1.js

# 3. Adicionar arquivos resolvidos
git add .

# 4. Continuar commit
git commit -m "v6.6: Merge com resolução de conflitos"

# 5. Push
git push origin main

───────────────────────────────────────────────────────────────────
↩️ SE QUISER DESFAZER (ANTES DO PUSH)
───────────────────────────────────────────────────────────────────

# Desfazer último commit (mantém mudanças):
git reset --soft HEAD~1

# Desfazer mudanças não commitadas:
git checkout -- .

# Ver histórico de commits:
git log --oneline

✅ DEPOIS DO PUSH
───────────────────────────────────────────────────────────────────

1. Aguardar 3-4 minutos
2. Verificar Cloudflare: https://dash.cloudflare.com
   • Pages → priobf25 → Deployments
   • Status: Success ✅

3. Testar:
   • https://priobf25.pages.dev/admin-v6.6.html

4. Verificar:
   • "Último commit" aparece no cabeçalho?
   • Widget "Histórico de Atualizações" está presente?
   • Editar produto registra no histórico?

───────────────────────────────────────────────────────────────────
🆘 TROUBLESHOOTING
───────────────────────────────────────────────────────────────────

❌ "Push rejected"
   → git pull origin main
   → git push origin main

❌ "Token in file"
   → Verificar se há tokens expostos
   → grep -r "ghp_" .
   → Remover tokens, commit novamente

❌ Deploy falhou no Cloudflare
   → Verificar logs: dash.cloudflare.com
   → Retry deployment
   → Verificar GITHUB_TOKEN configurado
