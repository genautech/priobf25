🎯 INÍCIO RÁPIDO - 3 PASSOS:
═══════════════════════════════════════════════════════════════

PASSO 1: Acesse o Sistema
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   1. Abra: admin-v6.3.html
   2. Você verá o Admin Panel atualizado
   3. Note o texto no topo: "v6.3 - CRUD Completo"

PASSO 2: Adicione Seu Primeiro Produto
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Opção A - Pelo Botão Verde:
   1. Vá para aba "Sugestões & Inovações"
   2. Clique no botão verde "Adicionar Produto" (topo direito)
   
   Opção B - Pela Aba Ferramentas:
   1. Clique na aba "Ferramentas"
   2. Procure o card "Adicionar Produto" (badge NOVO v6.3)
   3. Clique em "Novo Produto"
   
   4. Preencha o formulário:
      ➤ SKU: ELE-TEST-001
      ➤ Nome: Produto de Teste
      ➤ Categoria: Eletrônicos
      ➤ Subcategoria: Teste
      ➤ Fornecedor: Amazon
      ➤ Quantidade: 10
      ➤ Custo Base: 100.00
      ➤ Preço Mercado: 150.00
      ➤ Margem: 30
   
   5. Clique "Salvar Produto"
   6. ✅ Veja a notificação verde de sucesso!
   7. Produto aparece em TODAS as áreas automaticamente!

PASSO 3: Teste a Sincronização
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   1. Após adicionar o produto, navegue pelas abas:
      
      ✅ Dashboard → Veja estatísticas atualizadas
      ✅ Sugestões → Produto aparece na lista
      ✅ Catálogo → Disponível para clientes
      ✅ Análise → Métricas incluem novo produto
      ✅ Comparativo → Comparação de preços atualizada
   
   2. Tudo sincroniza AUTOMATICAMENTE! 🎉

═══════════════════════════════════════════════════════════════
📊 COMO IMPORTAR MÚLTIPLOS PRODUTOS:
═══════════════════════════════════════════════════════════════

PARA ADICIONAR 10, 50, 100+ PRODUTOS DE UMA VEZ:

1. Baixe o Template
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Admin → Ferramentas → "Baixar Template" (card 10)
   Arquivo: TEMPLATE-IMPORTACAO-PRODUTOS.csv

2. Preencha a Planilha
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Abra no Excel/Google Sheets
   Preencha uma linha por produto
   
   Colunas importantes:
   • sku: Código único (ex: ELE-0001)
   • nome: Nome do produto
   • categoria: Eletrônicos, Casa e Cozinha, etc.
   • custoBase: Quanto você paga (ex: 2000.00)
   • margem: Percentual de lucro (ex: 25)
   
   💡 Veja os 3 exemplos no template!

3. Importe
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Admin → Ferramentas → "Importar Excel/CSV" (card 9)
   Arraste o arquivo CSV OU clique para selecionar
   
4. Preview e Confirme
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Sistema mostra preview de TODOS os produtos
   Revise a tabela
   Clique "Confirmar Importação"
   
5. Veja os Resultados
   ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   Relatório mostra:
   ✅ Total de produtos
   ✅ Quantos importados com sucesso
   ❌ Quantos tiveram erro (com detalhes)
   
6. ✅ Produtos sincronizados em todas as áreas!
