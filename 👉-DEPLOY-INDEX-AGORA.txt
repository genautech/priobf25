═══════════════════════════════════════════════════════════════════
❌ PROBLEMAS COMUNS
═══════════════════════════════════════════════════════════════════

PROBLEMA: "nothing to commit"
─────────────────────────────────────────────────────────────────
CAUSA: Arquivo já foi commitado ou não foi modificado

SOLUÇÃO:
  • Verifique se você realmente modificou o index.html
  • Se sim, já está no GitHub (nada a fazer)


PROBLEMA: "Your branch is behind"
─────────────────────────────────────────────────────────────────
CAUSA: Versão local desatualizada

SOLUÇÃO:
  git pull origin main
  (depois faça o deploy novamente)


PROBLEMA: "authentication failed"
─────────────────────────────────────────────────────────────────
CAUSA: Git não está configurado

SOLUÇÃO:
  git config --global user.name "Seu Nome"
  git config --global user.email "seu@email.com"
  (depois faça o deploy novamente)


PROBLEMA: "Permission denied"
─────────────────────────────────────────────────────────────────
CAUSA: Token expirado ou sem permissão

SOLUÇÃO:
  • Use o Método 3 (Interface Web) - mais fácil
  • OU gere novo token no GitHub


PROBLEMA: Mudanças não aparecem no site
─────────────────────────────────────────────────────────────────
CAUSA: Cache do navegador

SOLUÇÃO:
  1. Aguarde 3-5 minutos (não apenas 2-3)
  2. Limpe cache: Ctrl+F5 (Windows) ou Cmd+Shift+R (Mac)
  3. Teste em aba anônima
  4. Teste em outro navegador
