ONDE É SALVO?
• 📍 LocalStorage do navegador
• 🔒 Apenas no seu computador
• 🔑 Nunca enviado para servidores externos

⚠️ IMPORTANTE:
O token GitHub é salvo localmente para conveniência.
Se usar computador compartilhado:

1. Use modo anônimo/privado, OU
2. Limpe depois no console:
   localStorage.removeItem('adminConfig_priobf25')
3. Revogue o token no GitHub depois de usar

┌──────────────────────────────────────────────────────────────────┐
│ 🔧 DEBUG (SE PRECISAR)                                           │

// Ver configuração salva

// Forçar salvamento
window.salvarConfiguracoesAdmin()

// Forçar carregamento
window.carregarConfiguracoesAdmin()

// Limpar tudo (reset)
localStorage.removeItem('adminConfig_priobf25')
