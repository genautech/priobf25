OPÇÃO B: Buscar Uma por Uma (melhor qualidade)
──────────────────────────────────────────────────

   → Escolha a melhor
   → Imagem é salva automaticamente

   → Você copia a URL da imagem
   → Cola quando solicitado

═══════════════════════════════════════════════════════════════
  🆘 PROBLEMAS COMUNS
═══════════════════════════════════════════════════════════════

Problema: "Botão Carregar Produtos está desabilitado"
→ Você não escolheu o método (SerpAPI ou Manual)
→ Escolha na seção "Passo 1"


Problema: "API Key inválida"
→ Verifique se copiou corretamente
→ Confirme o email no SerpAPI
→ Clique em "Testar API Key" para validar


Problema: "Limite de API excedido"
→ Você usou 100 buscas do mês
→ Aguarde próximo mês ou
→ Atualize para plano pago ou
→ Complete restantes manualmente


Problema: "Imagens não relacionadas ao produto"
→ Nome do produto pode estar confuso
→ Clique [🔍 Buscar Google] e escolha manualmente
→ Ou use [✏️ Manual] e cole URL correta


Problema: "Link do produto não abre"
→ URL pode estar inválida
→ Verifique no arquivo produtos-v6.1.js
