------------------------------
Solução: Configure token ou SSH
→ Use token do GitHub como senha
→ Ou configure chave SSH


Erro: "rejected non-fast-forward"
----------------------------------
Solução: Pull antes de push
→ git pull origin main --rebase
→ git push origin main


Erro: "no changes added"
-------------------------
Solução: Salve todos arquivos
→ Verifique se salvou tudo
→ git add . novamente
